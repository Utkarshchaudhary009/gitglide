# Source

A source represents a connected repository.

## Fields

| Field        | Type       | Description                                       |
| ------------ | ---------- | ------------------------------------------------- |
| `name`       | string     | The full resource name (e.g., 'sources/{source}') |
| `id`         | string     | Output only. The source ID                        |
| `githubRepo` | GitHubRepo | GitHub repository details                         |
